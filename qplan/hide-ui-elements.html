QPlan: Ẩn UI Elements trên Flow Page

Mục đích: Ẩn các thành phần giao diện không cần thiết trên trang flow editor của Langflow để có không gian làm việc tối giản hơn.

Quy trình thực hiện

Bước 1 — GitNexus Analysis: Sử dụng GitNexus MCP để phân tích codebase trước khi lập plan:
  • Đọc gitnexus://setup — lấy tổng quan repo (71,020 symbols, 134,588 relationships, 300 execution flows)
  • Đọc gitnexus://repo/langflow/context — stats, tools available, cluster/process resources
  • Đọc gitnexus://repo/langflow/clusters — xác định các functional areas liên quan (Ui, Components, Services)
  • Dùng gitnexus resources để xác định chính xác file paths, symbols, và process flows trước khi viết SRS
Bước 2 — Analyze: Dùng thông tin từ GitNexus để xác định chính xác các components, file paths, và dependencies.
Bước 3 — Write SRS + Wireframe: Lập tài liệu yêu cầu và wireframe.
Bước 4 — Lưu file plan: Ghi vào qplan/<feature-name>.html.
Bước 5 — Scope boundary: Đánh dấu in-scope / out-of-scope.
Bước 6 — Confirm: Trình plan cho user phê duyệt trước khi chuyển sang qcode.

Functional Requirements

FR1: Ẩn Top-Right Panel (Playground + Share + Deploy)

Vị trí: Góc phải trên của ReactFlow canvas — panel chứa các nút "Playground", "Publish", "Deploy".

Component: FlowToolbar → Panel position="top-right" trong src/frontend/src/components/core/flowToolbarComponent/index.tsx

GitNexus target: FlowToolbar symbol, processes: "render_flow_toolbar", cluster: Ui

CSS Target: Panel với className chứa !top-auto !m-2 trong #react-flow-id

ẨN

FR2: Ẩn Header Title Area

Vị trí: Thanh header trên cùng (48px), phần hiển thị tên flow / breadcrumb bên trái.

Component: appHeaderComponent/index.tsx → div chứa tên flow

GitNexus target: appHeaderComponent symbol, cluster: Ui

CSS Target: Phần tử div.hidden.pr-2.whitespace-nowrap.lg\\:inline-flex.lg\\:items-center trong header bar

ẨN

FR3: Ẩn MCP và Bundles Sidebar Buttons

Vị trí: Sidebar bên trái — các nút điều hướng MCP và Bundles.

Component: MemoizedSidebarTrigger → NAV_ITEMS map trong src/frontend/src/pages/FlowPage/components/PageComponent/MemoizedComponents.tsx

GitNexus target: NAV_ITEMS, sidebar-nav-mcp, sidebar-nav-bundles symbols

CSS Target: button[data-testid="sidebar-nav-mcp"] và button[data-testid="sidebar-nav-bundles"]

ẨN

Non-functional Requirements

  • NFR1: Các thay đổi chỉ ảnh hưởng đến CSS, không làm thay đổi logic component.
  • NFR2: Các element bị ẩn hoàn toàn (display:none), không chiếm không gian layout.
  • NFR3: Các element vẫn còn trong DOM (không bị xoá), có thể dễ dàng khôi phục.

Scope

In Scope:
  • Dùng GitNexus để phân tích codebase trước khi lập plan
  • CSS rules để ẩn 3 nhóm element trên flow editor page
  • Chỉ sửa CSS, không sửa logic component
  • Áp dụng cho mọi flow (global)
Out of Scope:
  • Không xoá component code
  • Không thêm tính năng toggle để hiện/ẩn
  • Không ảnh hưởng đến các page khác (chỉ flow page)
